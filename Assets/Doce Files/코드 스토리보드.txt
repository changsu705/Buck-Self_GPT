	2.	총 격발, 선택취소 ( 총성, 총의 철그럭 소리 )
	3.	아이템 카드 선택
	4.	아이템 카드 사용
	5.	피격

 Ui 설정
	1.	게임 시작 화면 ( 게임타이틀 및 로고, 로비 생성, 로비참가 )
	2.	로비 화면 ( 참가자 목록, 인원수 표시, 호스트는 게임시작 버튼 )
	3.	도움말 ( 느낌표 아이콘 클릭 시 게임 설명 - 조작법, 아이템 설명 등 )

게임 시스템 스토리보드
	1.	장전 시스템 표현법
	⁃	테이블 중앙에서 GM이 자동으로 장전 ( 허공에서 실린더에 실탄이 몇발 들어가는지 보여주는 방식 )
	⁃	다 쓴 총을 버리고 새로운 총을 꺼내면서 실탄 개수만 알려주는 방식
	2.	게임 시작시 가운데 있던 총이 랜덤 플레이어에게 자동으로 옮겨짐
	3.	아이템 카드가 덱에서 자동으로 나한테 들어옴 ( 무슨 카드인지 화면에 띄워주는 방식 )
	4.	카드를 좌클릭시 화면 중앙에 뜨고 한번 더 좌클릭시 사용 ( 사용시 중앙에 보내 모두에게 보이게 )
	5.	좌클릭으로 대상 지정 ( 화면이 지정대상에게 확대 되면서 조준 )
	6.	좌클릭으로 격발 ( 실탄시 총성과 탄배출, 공포탄시 틱 소리, 총을 확인 하는 애니 )
6-1. 피격 대상 ( 피격 사운드, 대상 표정 변경, 피격 시 모델 이펙트 )
	7.	격발 종료시 총이 다음 대상에게 자동으로 넘어감 ( 시계방향 )
	8.	대상 사망시 티비 화면이 깨지고  고개를 떨구는 식으로 표현 ( 관전 화면으로 이동 )
	9.	마지막 생존자가 나오면 승리 ( 화면 중앙에 글씨 표현 )
	10.	다시 로비화면으로 이동
